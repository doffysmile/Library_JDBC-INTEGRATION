# <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/java/java-original.svg" width="50"/> Library JDBC Integration

Projeto desenvolvido com o objetivo de praticar a integração entre aplicações Java e bancos de dados utilizando **JDBC (Java Database Connectivity)**.

## Objetivos

Aplicar conceitos de desenvolvimento back-end em Java, realizando operações de comunicação com banco de dados através da API JDBC.

Este projeto simula funcionalidades de um sistema de gerenciamento de biblioteca, permitindo o armazenamento e manipulação de informações de forma persistente.

## Tecnologias Utilizadas

* Java
* JDBC
* SQL

  
## Conceitos Praticados

* Conexão com banco de dados
* Consultas SQL
* Manipulação de dados
* Organização em camadas
* Tratamento de exceções
* Persistência de dados

## Funcionalidades

* Cadastro de livros
* Consulta de registros
* Atualização de informações
* Remoção de dados
* Integração entre Java e Banco de Dados
